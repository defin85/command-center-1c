# OpenSpec + Serena memory (минимум)

- OpenSpec (в репо) = "что должно быть": требования/решения/планы/spec/breaking changes. Вход: `openspec/AGENTS.md`.
- Serena memory = "как быстро работать": пути, команды, грабли; по возможности ссылки на файлы в репо.
- Неоднозначность или plan/proposal/spec → сначала OpenSpec; реализация/багфикс → контракты/доки + память для навигации.
