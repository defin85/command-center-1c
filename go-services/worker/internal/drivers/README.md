# Worker Drivers

Цель: вынести интеграционные способы доступа к 1С (RAS/OData/CLI/Agent/ibcmd) в самостоятельные драйверы,
а Go Worker оставить единственным engine, исполняющим операции по единому контракту.

На первом шаге драйверы могут делегировать в существующие реализации (для совместимости), но выбор operation_type
и общий lifecycle (timeline/metrics/events) должен быть единым и централизованным.
