# engineering-file-size Specification

## Purpose
Сделать кодовую базу более удобной для сопровождения и работы LLM/агентов за счёт уменьшения “монолитных” файлов: цель — держать исходные файлы в пределах ~700 строк (по `wc -l`) путём декомпозиции по модулям, без искусственных обходов.
## Requirements
### Requirement: Целевой лимит 700 строк для исходных файлов
Проект ДОЛЖЕН (SHALL) стремиться к тому, чтобы каждый исходный файл проекта (TypeScript/Go/Python, включая тесты) был декомпозирован так, чтобы размер файла не превышал **700 строк** (`wc -l`) для git-tracked файлов, за исключением явно определённых исключений (генерация, архивы, миграции и т.п.).

#### Scenario: Отчёт показывает отсутствие файлов >700 строк
- **GIVEN** определён список включаемых расширений и исключаемых путей/паттернов
- **WHEN** разработчик запускает отчёт по размерам исходных файлов
- **THEN** отчёт не содержит файлов с размером >700 строк (для включаемого набора)

### Requirement: Никаких “искусственных” обходов
Проект ДОЛЖЕН (SHALL) снижать размер файлов только за счёт **реального разноса по модулям** (выделение компонентов/хелперов/подмодулей/пакетов), без прагм/директив, “allowlist” исключений для отдельных файлов и без отключения проверок ради метрики.

#### Scenario: Рефакторинг выполняется через модульную декомпозицию
- **GIVEN** файл превышает 700 строк
- **WHEN** проводится рефакторинг
- **THEN** логика переносится в отдельные модули по ответственности
- **AND** поведение системы не изменяется

### Requirement: Frontend исходники декомпозированы до целевого размера (700 строк)
Проект ДОЛЖЕН (SHALL) выполнить рефакторинг frontend исходников (TypeScript/TSX, включая тесты) так, чтобы для включаемого набора файлов (см. `add-file-size-guideline-700`) не оставалось файлов размером более **700 строк** (`wc -l`), за счёт реального разноса по модулям.

#### Scenario: Отчёт по frontend не содержит файлов >700 строк
- **GIVEN** определены включаемые расширения и исключаемые пути/паттерны
- **WHEN** разработчик запускает отчёт по размерам файлов и ограничивает выборку на `frontend/**`
- **THEN** отчёт не содержит frontend файлов с размером >700 строк
