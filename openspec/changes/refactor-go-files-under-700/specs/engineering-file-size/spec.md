## ADDED Requirements
### Requirement: Go исходники декомпозированы до целевого размера (700 строк)
Проект ДОЛЖЕН (SHALL) выполнить рефакторинг Go исходников (включая тесты) так, чтобы для включаемого набора файлов (см. `add-file-size-guideline-700`) не оставалось файлов размером более **700 строк** (`wc -l`), за счёт реального разноса по файлам/подмодулям внутри пакетов.

#### Scenario: Отчёт по Go коду не содержит файлов >700 строк
- **GIVEN** определены включаемые расширения и исключаемые пути/паттерны
- **WHEN** разработчик запускает отчёт по размерам файлов и ограничивает выборку на `go-services/**`
- **THEN** отчёт не содержит Go файлов с размером >700 строк
