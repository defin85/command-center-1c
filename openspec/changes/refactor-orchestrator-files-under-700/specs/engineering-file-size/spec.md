### Requirement: Orchestrator (Python) исходники декомпозированы до целевого размера (700 строк)
Проект ДОЛЖЕН (SHALL) выполнить рефакторинг orchestrator исходников (Python, включая тесты) так, чтобы для включаемого набора файлов (см. `add-file-size-guideline-700`) не оставалось файлов размером более **700 строк** (`wc -l`), за счёт реального разноса по модулям.

#### Scenario: Отчёт по orchestrator не содержит файлов >700 строк
- **GIVEN** определены включаемые расширения и исключаемые пути/паттерны
- **WHEN** разработчик запускает отчёт по размерам файлов и ограничивает выборку на `orchestrator/**`
- **THEN** отчёт не содержит orchestrator файлов с размером >700 строк
